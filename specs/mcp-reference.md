# Model Context Protocol (MCP) Specifications

This project adopts the Model Context Protocol (MCP) for context exchange between the LLM and other tools/resources.

## Reference
The official specifications and documentation can be found at:
- **GitHub Repository**: [model-context-protocol/spec](https://github.com/model-context-protocol/spec)
- **Documentation**: [https://modelcontextprotocol.io](https://modelcontextprotocol.io)

## Integration
Merka Cloud components should adhere to the latest version of the protocol defined in the resources above.
